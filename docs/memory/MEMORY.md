## 设计规范（已确认）
- 暗色主题：bg=#0f1117, card=#1a1d27
- 颜色体系：紫色=#6366f1(齿轮1), 绿色=#34d399(齿轮2/滚子), 黄色=#fbbf24(渐开线/基圆/接触点), 橙色=#f97316(实际廓线)
- 凸轮线型规范：凸轮轮廓(实际廓线)=实线橙色，基圆=虚线[5,4]紫色，理论廓线=实线黄色
- 滚子从动件概念：理论廓线(黄)=pitch curve(外侧)=滚子中心轨迹，实际廓线(橙)=cam surface(内侧)=凸轮物理边界=理论廓线向内偏移rr
- 凸轮填充区域：滚子模式用actualProfile(内侧)，非滚子模式用profile
- 滚子从动件：滚子为主体(3px边框,0.2填充)，杆件弱化(1px,0.2透明)
- 接触点高亮：黄色圆点+白色中心，精确在actualProfile[degNow]位置
- 滚子中心：始终在固定导路位置(cx+e*scale, cy-totalS*scale)，不随凸轮旋转
- 半径标注：圆心到圆弧+单箭头，R和Rb用不同角度避免重合
- 旋转方向箭头：弯曲形状、白色系、尖锐
- 响应式布局适配手机电脑

## 可视化方案（已确认）
- 不用SolidWorks，直接用代码（Canvas/SVG）绘制
- 渐开线齿廓用数学公式精确生成
- 前期以2D为主，后期可选Three.js做3D
